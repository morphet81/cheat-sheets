Setup & Config

Set email git config --global user.email "email"
Set default editor git config --global core.editor vim
Set default branch name git config --global init.defaultBranch main
List all config git config --list
Create alias git config --global alias.co checkout

Create & Clone

Initialize new repo git init
Clone repository git clone <url>
Clone to folder git clone <url> <dir>
Clone specific branch git clone -b <branch> <url>
Shallow clone (depth 1) git clone --depth 1 <url>

Basic Workflow

Check status git status
Short status git status -s
Stage file git add <file>
Stage all changes git add -A
Stage hunks interactively git add -p
Commit with message git commit -m "message"
Stage & commit tracked git commit -am "message"
Amend last commit git commit --amend
Amend without editing git commit --amend --no-edit

List local branches git branch
List all branches git branch -a
Create branch git branch <name>
Switch branch git checkout <branch>
Create & switch git checkout -b <name>
Switch (new syntax) git switch <branch>
Create & switch (new) git switch -c <name>
Rename branch git branch -m <new>
Delete branch git branch -d <name>
Force delete branch git branch -D <name>
Delete remote branch git push origin -d <name>

Discard file changes git checkout -- <file>
Restore file (new) git restore <file>

Restore to location git show <branch>:path/file > dest/file
Unstage file git restore --staged <file>

Revert commit (new commit) git revert <sha>
